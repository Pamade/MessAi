Privacy Policy for MessAI

Last Updated: January 2025

Overview

MessAI is a browser extension that helps you generate AI-powered responses in Facebook Messenger. We are committed to protecting your privacy.

TL;DR: We don't collect ANY data. Everything stays on your device. Your API keys never leave your browser.

Data Collection and Storage

What We Collect

  • API Keys: Your Gemini, OpenAI, or Anthropic API keys (stored locally)
  • Settings: Your preferences (tone, prefix, response format)
  • History: Your prompts and responses (stored locally)
  • Templates: Custom message templates you create

Where Data is Stored

  • ALL data is stored locally in your browser using Chrome's storage API
  • NO data is sent to our servers - we don't have any servers
  • NO data is shared with third parties (except the AI providers you choose)

Third-Party Services

AI Providers

When you use this extension, your prompts are sent to:

  • Google Gemini API (if you use Gemini models)
  • OpenAI API (if you use GPT models)
  • Anthropic API (if you use Claude models)

Permissions Explained

  • storage: To save your settings, API keys, and history locally
  • tabs: To detect open Facebook/Messenger tabs
  • contextMenus: To provide right-click menu options
  • host_permissions: To inject the AI assistant into Facebook/Messenger

Data Security

  • API keys are stored encrypted in Chrome's local storage
  • No data leaves your device except API calls to your chosen AI provider
  • You can delete all data anytime from the extension settings

Your Rights

  • Access your data (it's all in your browser's local storage)
  • Delete your data (clear history in settings or uninstall extension)
  • Export your data (manually copy from history/templates)

Compliance

This extension complies with Chrome Web Store Developer Program Policies, GDPR, and CCPA.

By using this extension, you agree to this privacy policy.
